To execute findRichest.java, please enter the follow commands, which are listed below, into the command line. The program can read in from a passed in input file from the command line, or it will default to input.txt. The program is set to print the output to output.txt.

COMMAND LINE
To compile: javac findRichest.java
To run: java findRichest
To run with input file: java findRichest < inputfilename.txt
